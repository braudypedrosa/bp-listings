External search, mounted into the slot.

This page keeps bp-listings presentation-only. It mounts BPSearchWidget through renderSearchSlot, filters a local allListings array externally, then pushes the result back into the widget with setListings().

The map starts collapsed so the external search bar has room. Use the built-in map toggle to inspect marker updates.
