# Changelog

## 1.0.0

Initial public release of `bp-listings`.

### Included

- listings grid and interactive Leaflet map in a single widget
- card carousel, badges, favorite toggle, and listing click callbacks
- built-in sorting, pagination, map toggle, and fullscreen map action
- search-slot integration for mounting an external search UI above the listings grid
- runtime APIs for replacing listings, paging results, panning to listings, and destroying the widget
- helper utilities for matching `searchData` payloads against listing data
